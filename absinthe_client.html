    AbsintheClient

    Elixir library to perform server-side queries on a local Absinthe-based GraphQL API. WIP adaption of Absinthe.Phoenix.Controller to be used with LiveView.

    Usage:

    defmodule MyApp.Web.WidgetsLive do
  use MyApp.Web, :live_view

  use AbsintheClient, schema: Bonfire.GraphQL.Schema, action: [mode: :internal]

  def mount(params, session, socket) do
    widgets = awesome_widgets(socket)
    IO.inspect(widgets)

    {:ok, socket
    |> assign(
      widgets: widgets
    )}
  end

  # notice we use snakecase rather than camelcase
  @graphql """
    {
      awesome_widgets
    }
  """
  def awesome_widgets(socket), do: liveql(socket, :awesome_widgets)

end
