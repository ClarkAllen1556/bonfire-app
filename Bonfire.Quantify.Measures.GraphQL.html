    map_ok_error(items, func) :: {:ok, any()} | {:error, term()}
when items: [Map.t()], func: (Map.t(), any() -> {:ok, any()} | {:error, term()})

    Iterate over a set of elements in a map calling func.

    func is expected to return either one of {:ok, val} or {:error, reason}. If {:error, reason} is returned, iteration halts.
