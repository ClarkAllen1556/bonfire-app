    Bonfire.Federate.ActivityPub.APPublishWorker (bonfire v0.1.0-cooperation-beta.44)

    Module for publishing ActivityPub activities.

    Intended entry point for this module is the __MODULE__.enqueue/2 function provided by ActivityPub.Workers.WorkerHelper module.

    Note that the "context_id" argument refers to the ID of the object being federated and not to the ID of the object context, if present.
